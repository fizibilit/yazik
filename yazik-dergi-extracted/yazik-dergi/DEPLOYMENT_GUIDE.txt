ADIM 1: GitHub Repo Oluştur

2. Terminal/Git Bash'ta:

   cd yazik-dergi

   cp /mnt/user-data/outputs/master.json ./data/master.json

master.json Format:

    {
      "cilt_no": 1,

            {

}

Temel İşlemler:
- Veri ekle: master.json düzenle → push
- Test et: localhost:8000 (local server)
- Deploy: GitHub push → Netlify otomatik
- Hata: Browser F12 → Console → Hata oku

═══════════════════════════════════════════════════════════════════════════════
📞 DESTEK
═══════════════════════════════════════════════════════════════════════════════

Hata Giderme:
1. Browser cache temizle (Ctrl+Shift+Del)
2. Console'u aç (F12)
3. Network sekmesi: master.json 200 OK mi?
4. Responsive mode (F12 → Ctrl+Shift+M)

Netlify Hataları:
- Deploy başarısız: Repo'da en az index.html olması gerek
- Site açılmıyor: netlify.toml'daki publish dizini kontrol et
- DNS hatası: Subdomain ayarlarını kontrol et

Veri Hataları:
- "Yazılar görülmüyor": JSON syntax hatası (comma, bracket)
- "Dropdown boş": ciltler array'i boş
- "Grid bozuk": CSS override (F12 → Styles)
